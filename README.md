# 🌐 voytek.ai

Personal portfolio website for Voytek Kopcewicz — IT Infrastructure specialist, AI automation builder, and home lab enthusiast based in Atlanta, GA.

## 🔗 Live Site
[voytek.ai](https://voytek.ai)

## 🛠️ Built With
- HTML, CSS, vanilla JavaScript
- No frameworks, no templates — written from scratch
- Hosted on a Raspberry Pi 5 running Nginx
- Served via Cloudflare Zero Trust tunnel

## ✨ Features
- Custom green glowing cursor with pulse animation
- Smooth scroll navigation
- Responsive design with mobile hamburger menu
- Fade-in animations on scroll
- Dark cyberpunk aesthetic

## 🖥️ Infrastructure
- **Server:** Raspberry Pi 5 (Ubuntu) with Pironman5 case
- **Web Server:** Nginx
- **Tunnel:** Cloudflare Zero Trust
- **Storage:** 458GB SATA SSD
